def IsFiniteUnionOfPointsAndIntervals [LinearOrder R] (A : Set R) : Prop :=
  ∃ L : List (Set R),
    (∀ I ∈ L,
      (∃ a, I = ({a} : Set R)) ∨
      (∃ a b, I = Ioo a b) ∨
      (∃ a, I = Iio a) ∨
      (∃ a, I = Ioi a) ∨
      I = univ) ∧
    A = ⋃ I ∈ L, I

Explanation:
Defines what it means for a subset of the line `R` to be a finite union of basic one-
dimensional pieces. The allowed pieces are singleton points, open bounded intervals,
open rays, and the whole universe. This is the one-dimensional finiteness condition used
in o-minimality.
